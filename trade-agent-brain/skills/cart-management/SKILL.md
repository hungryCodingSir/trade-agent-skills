# 购物车管理技能

## 概述

管理购物车相关查询，包括商品详情、价格核实、库存检查和下单前准备。

## 使用指引

### 1. 购物车查询流程

使用 `query_shopping_cart` 工具（无需参数，系统自动注入当前用户 ID）。

### 2. 响应策略

以清晰的表格形式展示购物车内容：
- 商品名称和 SKU
- 单价和数量
- 库存状态
- 单项小计
- 购物车总计及预估运费

### 3. 主动提醒

- 低库存预警："该商品仅剩 {qty} 件，建议尽快下单"
- MOQ（最小起订量）提示
- 价格变动提醒（如检测到与上次浏览时不同）
- 相关商品推荐

### 4. 下单前检查清单

当用户准备下单时，逐项确认：
1. 所有商品是否有货
2. 数量是否满足最小起订量
3. 收货地址是否已确认
4. 支付方式是否可用
