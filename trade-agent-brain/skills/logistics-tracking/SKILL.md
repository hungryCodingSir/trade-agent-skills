# 物流追踪技能

## 概述

提供跨境物流的实时追踪服务，包括多承运商追踪、海关清关状态和到达时间预估。

## 使用指引

### 1. 判断追踪场景

- **实时追踪**：用户想知道当前位置/状态 → 使用 `query_shipping_info` 工具
- **到达预估**：用户问"什么时候到" → 根据物流数据计算
- **异常预警**：货物延迟或滞留 → 主动提醒
- **清关状态**：海关通关进度 → 从物流轨迹中提取

### 2. 多段物流处理

跨境运输通常有多个段落：
1. **国内揽收**（产地仓库 → 出发港口）
2. **国际运输**（海运/空运）
3. **目的国清关**（海关查验放行）
4. **末端派送**（本地快递 → 买家地址）

始终识别货物当前处于哪个段落。

### 3. 异常检测与主动预警

自动标记以下情况：
- 物流信息超过 5 天未更新
- 清关超过 3 个工作日未放行
- 派送失败（无人签收）
- 超过预计到达日期仍未送达

检测到异常时：
- 解释可能原因（如海关扣留、天气延误）
- 建议后续操作（联系承运商、补充文件）
- 主动提出发送邮件通知相关方

### 4. 响应格式

```
📦 物流追踪 — 订单 {order_no}
━━━━━━━━━━━━━━━━━━━━━━
🚢 承运商: {carrier} | 运单号: {tracking_no}
📍 当前状态: {status}
📅 预计到达: {eta}

物流轨迹:
  ✅ 2024-01-15 出库发货（深圳仓库）
  ✅ 2024-01-16 到达港口（深圳蛇口港）
  🔄 2024-01-18 海运中（预计7天）
  ⏳ 待清关（洛杉矶港）
  ⏳ 待派送
```

### 5. 卖家专属功能

- 展示待发货订单列表
- 批量物流状态概览
- 承运商时效对比分析
