# 海关清关技能

## 概述

为跨境电商提供海关和贸易合规指导，
包括 HS 编码分类、关税估算和单证要求。

## 适用范围

本技能由主 Agent 直接处理，**无对应子智能体**：
- ✅ HS 编码分类咨询（单个商品）
- ✅ 单个订单的关税估算
- ✅ 标准文件清单和合规指引
- ✅ 常见清关问题的应答

> 清关咨询属于知识问答类任务，主 Agent 基于本 Skill 即可回答。
> 如果用户的清关问题涉及具体订单的物流卡关，需配合使用 `query_shipping_info` 查询物流状态。

## 边界判断

- "电子产品出口到美国需要什么文件" → 知识问答，**主Agent自行处理**
- "ORD001 卡在海关了怎么办" → 查物流 + 给建议，**主Agent自行处理**（配合 `query_shipping_info`）
- "分析我最近10个订单的清关通过率" → 多订单聚合分析，**委派 analytics-specialist**


## 使用指引

### 1. HS 编码分类

当用户询问商品归类时：
- 询问商品描述、材质和用途
- 提供最可能的 HS 编码及置信度
- 注意：务必建议用户向海关官方确认
- 附上目的国对应的税率

### 2. 关税计算

计算公式：`关税 = CIF 价值 × 关税税率`
其中 CIF = 商品成本 + 保险费 + 运费

提供以下信息：
- 预估关税金额
- 增值税/消费税（如适用）
- 优惠贸易协定减免（如 RCEP、USMCA）

### 3. 所需文件清单

标准跨境运输文件：
- [ ] 商业发票（Commercial Invoice）
- [ ] 装箱单（Packing List）
- [ ] 提单/空运单（Bill of Lading / Air Waybill）
- [ ] 原产地证明（Certificate of Origin）
- [ ] 进口许可证（如适用）
- [ ] 产品安全认证（CE/FCC/CCC）

### 4. 常见问题与处理

| 问题 | 可能原因 | 处理方式 |
|------|----------|----------|
| 海关扣留 | 文件缺失 | 尽快补交所需文件 |
| 关税争议 | HS 编码错误 | 申请归类复核 |
| 没收通知 | 属于禁运商品 | 联系报关行处理 |
| 放行延迟 | 高价值货物抽检 | 提供价值证明材料 |

### 5. 免责声明

每次回复末尾附加：
"以上信息仅供参考，具体清关要求请以目的国海关最新规定为准。建议重要货物咨询专业报关行。"
