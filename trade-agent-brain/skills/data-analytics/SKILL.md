# 数据分析技能

## 概述

为跨境电商运营生成数据分析报告和业务洞察。
支持卖家绩效、买家行为和平台级数据分析。

## 使用指引

### 1. 分析类型

| 类型 | 适用角色 | 关键指标 |
|------|----------|----------|
| 销售报告 | 卖家/管理员 | 收入、订单量、客单价、转化率 |
| 退货分析 | 卖家/管理员 | 退货率、退货原因、退货成本 |
| 供应商评分卡 | 管理员 | 准时交付率、质量合格率、响应速度 |
| 买家画像 | 卖家 | 购买频次、客户生命周期价值、偏好 |
| 平台仪表盘 | 管理员 | GMV、活跃用户数、纠纷率 |

### 2. 报告生成流程

1. 根据用户查询识别分析类型
2. 通过 MCP 工具获取相关数据
3. **使用文件系统将报告写入结构化文档**
4. 向用户展示关键发现摘要
5. 保存完整报告供后续查阅

### 3. 可视化建议

展示数据时建议合适的图表类型：
- 时间趋势 → 折线图
- 分类对比 → 柱状图
- 构成占比 → 饼图
- 相关性分析 → 散点图

### 4. 可执行洞察

每份报告结尾应包含：
- **前 3 大发现** — 最重要的数据变化
- **行动建议** — 具体可执行的改进措施
- **风险预警** — 负面趋势的指标

### 5. 委派策略

对于复杂的多维度分析，委派给 data-analytics 子智能体处理，
避免主智能体的上下文窗口被中间数据淹没。
